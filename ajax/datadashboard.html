Contexto & problema
Laboratoria cuenta con un Learning Management System que tiene lecturas y ejercicios para las alumnas del bootcamp. Las Training managers (TMs) son las personas encargadas de analizar y evaluar el avance, no obstante, actualmente no cuentan con una interfaz que les permita ver esta data.
El reto
Crear una interfaz que muestre el progreso de las alumnas de forma visual y ordenada.
La solución
Después de entrevistar Alejandra y de realizar múltiples testings, se creó una plataforma con tres secciones: general, progreso y alumnas.
Lo que aprendí
Con este reto reforcé mis conocimientos sobre HTML5, comprendí la lógica de CSS3, aprendí funciones y distintos métodos de arrays en Javascript. A nivel de UX, desarrollé entrevistas y aprendí a crear un User Persona.
