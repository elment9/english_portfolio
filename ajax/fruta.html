Contexto
Crear un marketplace fue nuestro primer desafío en el track de UX Design de Laboratoria. Para este reto decidí elegir a Fruta La Grande, un emprendimiento que se dedica a vender fruta orgánica bajo la modalidad delivery.
El problema
Actualmente atienden todos los pedidos a través de Facebook y Whatsapp, no existe un canal automático. Esto perjudica tanto al cliente como al usuario. A ello se suma que los clientes escriben en diferentes horarios y tienen preguntas frecuentes que necesitan ser atendidas.
El reto
Crear un e-commerce funcional con un flujo simple y amigable que brinde seguridad a los usuarios.
La solución
Durante dos semanas se realizó una árdua investigación enfocada en descubrir las principales motivaciones y frustraciones de los usuarios de Fruta La Grande. Así, se creó una tienda virtual funcional que emula, a través de una UI minimalista, el proceso de compra en una tienda.
Las categorías se validaron luego de realizar cardsorting. Se agregó el pago a contraentrega, considerado por los usuarios como la forma de pago más segura, y el pago con tarjeta. Para responder a las dudas, se implementó una sección de preguntas frecuentes, además se creó una mascota que acompaña a los compradores durante el proceso de compra.
Lo que aprendí
Para este proyecto, Laboratoria nos dio un presupuesto límite (x puntos por cada técnica), aprender a distribuir los recursos fue un aprendizaje importante. Se suma a ello el aprender sobre e-commerce y técnicas como cardsorting, affinity map, benchmark, customer journey map, what if, etc.
